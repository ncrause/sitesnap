API Usage

SiteSnap provides a free API for invoking website photographs. When invoked, it returns either a lossless PNG file, or a response structure in JSON. You can tell which has been returned by either inspecting the Content-Type header, or by the HTTP status code - 200 will always only be returned if a successful photograph was taken.

Limits
